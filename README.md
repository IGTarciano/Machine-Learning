# Machine-Learning
Repositório para modelos de aprendizado de maquina

Este repositório reúne uma série de modelos de aprendizado de máquina desenvolvidos por mim com foco em previsão de demanda e análise de séries temporais. Utilizando abordagens como SARIMAX e regressão com variáveis exógenas, os modelos foram treinados e avaliados com base em dados históricos de vendas, faturamento e estoque.

O projeto inclui:
- Pré-processamento de dados com transformação logarítmica
- Separação entre treino e teste com validação temporal
- Modelagem com SARIMAX incorporando variáveis exógenas
- Avaliação de desempenho com métricas como erro absoluto e percentual
- Exportação dos resultados para análise em Power BI
  
O objetivo principal é oferecer uma base sólida para aplicações reais de previsão de vendas, planejamento de estoque e apoio à tomada de decisão em ambientes comerciais. Todos os scripts estão organizados e documentados para facilitar a reprodução e adaptação a outros contextos.
